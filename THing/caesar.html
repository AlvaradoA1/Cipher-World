Caesar Cipher

Hint! Hints: An easy way to generate the key is to convert each letter into its numeric equivalent starting with ‘A’=0 to ‘Z’ = 25. Applying a shift of 5 is equivalent to adding 5 to the numeric equivalent. For instance, Original:A → Encoded: ‘F” (‘A’+5 = 0+5 = 5 = ‘F’). You only have to test a few letters with the shifted key to test if the shift makes sense! For instance, there are no English words that have the bigram (letters next to each other) ‘QY’.

Caesar Cipher is a simple substitution cipher where each letter of the alphabet is shifted by a constant number of spaces to create an encoding key. For instance, given the encoded letter “C” and a shifting factor of 2, then the original letter is “A.” Another example would be an encoded letter “A” and a shifting of factor of 16, which would yield the original letter “ K.” Thus, when solving these puzzles, attempt to find the shifting factor, and then apply the key to easily decrypt the rest of the message.
